## تدفق إضافة شحنة

```
[باركود / يدوي] ← رقم الشحنة
       ↓
اسم العميل + جوال العميل
       ↓
اسم المتجر
       ↓
العنوان الوطني (يُحوَّل تلقائياً لإحداثيات)
       ↓
نوع الدفع: مدفوع / COD
       ↓
صورة البوليصة (اختياري)
```

> **العنوان الوطني:** يُرسَل لـ Nominatim (OpenStreetMap) ليعود بالإحداثيات الدقيقة

---

## تدفق التوصيل — DeliveryFlow

```
┌─────────────────────────────────────────┐
│  مرحلة: استلام الشحنة (receipt)        │
│  يظهر: بيانات الشحنة + خريطة الوجهة   │
│  زر: 🚀 بدء الرحلة                     │
└──────────────┬──────────────────────────┘
               │ [طلب إذن الموقع]
               ▼
┌─────────────────────────────────────────┐
│  مرحلة: في الطريق (traveling)          │
│  يظهر: خريطة حية (السائق → الوجهة)   │
│  GPS يُرسَل لـ Firebase كل ثوانٍ      │
│  زر: 📍 وصلت للعنوان                  │
└──────────────┬──────────────────────────┘
               │
               ▼
┌─────────────────────────────────────────┐
│  مرحلة: وصل (arrived)                 │
│  يوقف GPS + يحدّث Firebase            │
│  خياران:                               │
│  ✅ تم التوصيل                         │
│  ❌ تعذر التوصيل                       │
└──────────────┬──────────────────────────┘
               │
    ┌──────────┴──────────┐
    ▼                     ▼
✅ موصّل            ❌ تعذر
                    أسباب التعذر:
                    • لم يرد العميل
                    • عنوان خاطئ
                    • رُفض الاستلام
                    • العميل طلب التأجيل
                    • غيره (نص إلزامي + صورة)
```

## خريطة التتبع — تفاصيل تقنية

- **المكتبة:** Leaflet.js 1.9.4 عبر CDN (داخل `srcdoc` في iframe)
- **الخرائط:** OpenStreetMap tiles
- **المؤشرات:** نقطة خضراء نابضة لكل سائق نشط
- **Popup:** اسم السائق + رقم الشحنة + العنوان + الشحنات المعلقة
- **التحديث التلقائي:** `key` يتغير مع كل تحديث موقع → iframe يُعاد تحميله
- **شرط النشاط:** `active === true` + آخر تحديث أقل من دقيقتين

## GPS وFirebase

```js
// السائق يكتب موقعه (كل تحديث GPS)
cloud.setDriverLocation(driverId, {
  lat, lng, acc, ts,
  name, active: true,
  shipmentNo, destAddress
})

// الإدارة تستقبل المواقع فور تغيرها
cloud.listenDriverLocations(fbLocs => {
  setLocations(prev => ({ ...prev, ...fbLocs }))
})

// عند الوصول
cloud.clearDriverLocation(driverId) // → { active: false }
```
